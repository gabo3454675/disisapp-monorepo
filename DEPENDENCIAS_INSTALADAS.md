# Dependencias ya Instaladas ✅

Las siguientes dependencias ya están instaladas en el proyecto:

## Dependencias de Producción
- `@nestjs/passport` - Framework de autenticación
- `passport-jwt` - Estrategia JWT para Passport
- `passport-local` - Estrategia local para Passport
- `bcrypt` - Para hash de contraseñas
- `@nestjs/jwt` - Módulo JWT de NestJS
- `class-validator` - Validación de DTOs
- `class-transformer` - Transformación de objetos

## Dependencias de Desarrollo
- `@types/bcrypt` - Tipos TypeScript para bcrypt
- `@types/passport-jwt` - Tipos TypeScript para passport-jwt
- `@types/passport-local` - Tipos TypeScript para passport-local

## Comando de Instalación (por si necesitas reinstalar)

Si necesitas instalar las dependencias nuevamente, ejecuta:

```bash
cd apps/server
pnpm install
```

O desde la raíz del monorepo:

```bash
pnpm install
```
